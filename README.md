项目功能

	高清视频会议和语音通话和文本聊天。
	多达4人免费视频会议在同一时间在同一房间。
	自己创建房间简单分享，点击链接即可入会。
	消息实时提醒。
	无需注册。

	1.本http service 基于nodejs的环境下运行，前提必须安装node.js的编译环境；
